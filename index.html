Welcome to the Student Club Micro Course

Learn HTML fundamentals in 3 concise sessions

This micro course is designed to introduce you to the basics of HTML and web development. Over the course of three sessions, you will learn the fundamental concepts and skills needed to create simple web pages.

By the end of this course, you will be able to build and style your own basic web pages using HTML and CSS.

This course is ideal for beginners who want to start building their own websites.

Join us to explore the exciting world of web development!

  1. Learn the basic structure of an HTML document
  2. Understand common HTML tags and their purpose
  3. Write semantic HTML to structure content effectively
Do I need prior experience?
No prior coding experience is required.
How long is the course?
The course runs over three short sessions.
